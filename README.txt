======= CRM BACKEND =======

Après avoir cloné ou téléchargé le projet faire:

- npm install  (afin d'installer les dépendances nécessaires)

- Importer la base de données database.sql présent dans le répertoire /src/db

- Modifier les paramètres de connexion MySQL (nom d'utilisateur et mot de passe) dans le fichier data-source.ts

- Pour lancer le projet, rendez-vous dans le répertoire du projet et taper: npm run start:dev
